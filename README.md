# Breast Cancer Classification using Neural Network

This project focuses on building a machine learning model to classify breast cancer tumors as malignant or benign.

## 📌 Project Overview
In this project, I used a neural network to predict whether a tumor is malignant or benign based on medical features from the dataset.

## ⚙️ What I did
- Loaded and explored the breast cancer dataset
- Performed data preprocessing and feature scaling
- Built a neural network model using TensorFlow/Keras
- Converted prediction probabilities into class labels
- Evaluated the model using accuracy and confusion matrix
- Tested the model on new input data

## 📊 Results
- Model Accuracy: ~94%
- Successfully classified tumors with high confidence

- Scikit-learn
- TensorFlow / Keras
